## ¿Qué es?
Jinx es una instancia configurada de Hermes Agent (Nous Research) corriendo
en servidor Debian self-hosted, con memoria episódica persistente, routing
inteligente de modelos y conexión a servicios externos reales.

## Plataformas conectadas
| Plataforma | Estado | Uso |
|-----------|--------|-----|
| Telegram | ✅ Activo | Home base, comandos, notificaciones |
| Home Assistant | ✅ Activo | IoT — TV, PC, luces |
| GitHub | ✅ Activo | Repos, PRs, CI/CD, despliegues |
| Discord | 🔗 Conectado | Canal de respaldo |

## Intelligence Stack
- **Memory:** Mnemosyne — stores episódico y working memory
- **Routing:** Criterio costo/eficiencia (Gemini Flash Lite primary)
- **Reasoning:** Chain-of-thought interno
- **Proactive Engine:** Cronjobs automáticos y monitoreo continuo
